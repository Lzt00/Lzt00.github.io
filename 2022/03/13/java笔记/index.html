一、什么是JDK、JRE（Java是解释性语言）

JDK=JRE+开发工具集合（Javac、Java等）

JRE（运行.class的环境即运行Java程序）=JVM+javaSE的核心类库

总结：JDK包含了开发和运行Java的所有功能，故只需安装一个JDK即可

四、Java数据类型

1. 基本数据类型

(1) 数值型

整数型：byte[1] short[2] int[4] long[8]

浮点型：float[4] double[8]

(2)字符型

**char[2]**，存放单个字符’a’

(3) 布尔型

boolean[1],存放true和false

2. 引用数据类型

类(class) 接口(interface) 数组[]

3. 整型使用细节

  1. 整型默认为int类型，若要声名为long类型则需在数据后面加L

  2. bit(位)是计算机中的最小存储单位。byte(字节)是计算机中的基本存储单位1byte=8bit 即1字节=8位

4. 浮点型使用细节

  1. 浮点型常量默认为double型，若要声名为float类型则需在数据后面 加F

  2. 浮点型常量有两种表现形式：

    十进制形式：5.2 5.21

    科学计数法形式：5.12e10 5.21e-1

  3. 通常情况下，应使用double类型，因为更精确

  4. 浮点数使用陷阱：2.7和8.1/3的比较(详见doubledatil.java)

  5. 小数尾数部分可能丢失，造成精度损失(小数都是近似值)。

5. 字符型使用细节(char1.java)

  1. 字符常量是用单引号括起来的单个字符。如char c=’a’ char c=’李’

    注意char=”9”，用双引号括起来则是字符串。

  2. 也可以使用转义符’\‘，将其后的字符转变为特殊字符型常量。如：char c=’\n’ 表示换行。

  3. 在Java中，char的本质是一个整数，在输出时，会按照对应的unicode字符输出

  4. char 类型是可以参与运算的，相当于一个整数，在参与运算时系统将会把字符转化成对应的编码数字来进行运算。

  5. 可以给char赋一个整数，输出的会是与之对应的字符。

  6. 总结：char类型若在输出时参与运算结果会以整数类型输出。

6. 布尔型（boolean）

仅能用来存放true和false

十五、 进制转换

1. 二进制转十进制

从最低位开始，将每个位上的数提取出来，乘以2的（位数-1）次方，然后求和

0b1011 = 1 * 2的1-1次方+1*2的2-1次方+1*2的4-1次方

2. 八进制转十进制

从最低位开始，将每个位上的数提取出来，乘以8的（位数-1）次方，然后求和

0234 = 4*8的0次方+3*8的1次方+2*8的2次方

3. 十六进制转十进制

从最低位开始，将每个位上的数提取出来，乘以16的（位数-1）次方，然后求和

0x23A = 10*16^0+3*16^1+2*16^2

4. 十进制转二进制

不断除2，余数倒过来

5. 十进制转八进制

不断除8，余数倒过来

6. 十进制转十六进制

不断除16，余数倒过来
